# Contexte métier SYCEBNL

## Domaine

Le projet traite la comptabilité d'organisations au format SYCEBNL.

## Axes fonctionnels

- journaux;
- grand livre;
- balance;
- clôture;
- amortissements;
- TVA;
- états financiers;
- contrôle de cohérence métier.

## Règle d'or

Les agents métier doivent valider la cohérence comptable avant toute implémentation technique.

## Interdits pour les agents métier

- écrire du code Flutter;
- modifier l'architecture de présentation;
- contourner les règles comptables pour simplifier l'UI.
